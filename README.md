## BigQuery-Practical Overview
This project contains SQL solutions for a Retail Sales Dataset using Google BigQuery. It is part of a practical data analytics exercise focused on building foundational SQL skills such as filtering, aggregation, grouping, and conditional logic.

The goal is to analyze transactional data and extract meaningful insights using structured queries.

## Dataset Description

The dataset includes retail transaction data such as:

- Customer details (demographics)
- Product categories
- Transaction amounts
- Quantity and price per unit

## Key SQL Concepts Covered
- WHERE filtering
- Aggregate functions (SUM, AVG)
- GROUP BY and HAVING
- DISTINCT values
- CASE statements
- Subqueries
- Conditional aggregation

## Summary

This project demonstrates how SQL can be used to transform raw retail data into actionable insights using other tools(BigQuery). By applying filtering, aggregation, and conditional logic, meaningful patterns and trends can be identified to support business decision-making.
